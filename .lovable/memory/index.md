# Project Memory

## Core
BeBright Planner — workforce mgmt platform for LED installation. Electric Blue #0EA5E9, dark slate bg #0F172A.
Inter font. AED currency, 24h time, DD/MM/YYYY. Status: green=present, amber=traveling, red=absent, blue=planned, purple=overtime.
Lovable Cloud enabled. React+Vite (not Next.js). Android app is separate — web portal is admin/manager only.
Web portal does NOT collect GPS — only displays GPS data from Android app.
